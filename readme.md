# GIF-to-ASCII Converter — Веб-сервис конвертации анимаций

Веб-приложение для интерактивной конвертации GIF-анимаций в символьную ASCII-графику в реальном времени. Проект имеет две версии: персональную кастомную версию (разработанную в качестве уникального подарка на 14 февраля) и универсальную Production-версию для открытого использования.

## 🛠 Стек технологий

*   **Backend:** Node.js, Express
*   **Frontend:** TypeScript, React, Tailwind CSS, Vite
*   **Инструменты обработки:** Puppeteer (для рендеринга), FFmpeg (для работы с кадрами видео/анимаций)
*   **Infrastructure:** Docker, Nginx (Reverse Proxy)

---

## ✨ Основной функционал (Features)

*   **Гибкая кастомизация интерфейса:** Возможность динамически менять цветовую палитру и тему оформления панели.
*   **Управление воспроизведением:** Интерактивный плеер с возможностью поставить ASCII-анимацию на паузу или изменить скорость воспроизведения.
*   **Настройка качества (Разрешение):** Изменение плотности символьной матрицы (можно поднять или опустить количество символов в кадре для детальности изображения).
*   **Информационный оверлей:** Нативный счетчик FPS (частоты кадров) для контроля плавности рендеринга.
*   **Экспорт результатов:** Возможность скачать готовую сконвертированную анимацию обратно в формате `.gif`.

## ⚙️ Установка и локальный запуск

Развертывание проекта максимально упрощено и не требует ручной настройки переменных окружения (`.env`). Всё окружение вместе с необходимыми системными зависимостями (`ffmpeg` и `chromium` для `puppeteer`) уже настроено внутри Docker-контейнеров.

### 1. Клонирование репозитория
```bash
git clone https://github.com/arcwagner1337/ascii-converter
cd ascii-converter
```

### 2. Сборка и запуск системы
Выполните команду из корневой папки проекта:
```bash
docker-compose up -d --build
```

После того как Docker поднимет контейнеры и Nginx настроит проксирование портов, приложение будет доступно локально по адресу `http://localhost`.
